## -*- coding: utf-8 -*- Naphtaline: BUG Tracking made REALLY simple (and beautiful)
Naphtaline
Bug tracking made really simple
Nouveau projet
En utilisant ce formulaire, vous pourrez créer votre propre projet sur Naphtaline.
Vous ne savez pas ce qu'est Naphtaline ? Cliquez ici pour en savoir plus.
Première étape : les caractéristiques de votre projet
Le nom de votre projet
Vous ne pouvez pas choisir un nom vide.
Le "nom web" de votre projet
Le "nom web" est une version courte du nom qui sera inclue dans l'adresse. Elle ne peut contenir que des caractères alphanumériques en minuscule (a-z0-9).
Par exemple, un project appelé "Mon projet au nom beaucoup trop long" avec le "nom web" de "monprojet" sera accessible à l'adresse http://projects.naphtaline.net/myproject
Le nom web que vous avez choisit contient des caractères non autorisés.
Le nom web que vous avez choisit est déjà utilisé par un autre projet.
Vous devez choisir un nom web.
La langue de votre projet
Choisissez ici la langue que vous désirez pour l'interface de votre projet.
Seconde étape : votre compte administrateur
Login
Vous devez choisir un login
Mot de passe
Vous devez entrer un mot de passe.
Les deux mots de passe que vous avez entré sont différents.
Entrez votre mot de passe
Ré-entrez votre mot de passe (pour vérification)
E-mail
Vous devez entrez une adresse email valide.
Dernière étape : créer le projet

Merci de patienter pendant la création du projet.
Une erreur est survenue. Merci de bien vouloir réessayer plus tard.
Votre projet a été créé avec succès. Un mail récapitulatif vous a été envoyé.
Vous allez être redirigé vers la page de votre projet. Si vous n'êtes pas redirigé dans les 15 secondes, cliquez ici.
